## Run Locally

#### Get started

1. **Clone the repo:**
```bash
  $ git clone https://github.com/KuldeepSinghRathore/MQuiz.git
```
2. **Install required node modules:**
```bash
  $ npm install
```
3. **Start the dev server:**
```bash
  $ npm start
```
Open the `localhost` link in the browser.
